## Project Overview
This project analyzes retail sales data to identify key business insights such as total revenue, category performance, and sales distribution.

## Business Questions Answered
1. How much are we selling? (Total Sales)
2. Which category performs best?
3. What is the percentage contribution of each category?

## Tools Used
- SQL Server (SSMS)
- Power BI
- Excel

## Key Insights
- Total Sales: 1.45M
- Grocery is the highest performing category.
- Sales contribution is evenly distributed across categories (~23%-27%).

- Dataset (CSV)
